Core NLP Modeling: Classification and Embeddings

Remark

These lecture notes are in early access and will continue to evolve based on classroom experience and feedback.

3. Core NLP Modeling: Classification and Embeddings#

With the foundations of text preprocessing and vectorization established in the previous chapters, we are ready to build and evaluate models that solve real NLP tasks. This chapter focuses on supervised text classification — one of the most common and practically important NLP problems — and on the richer representation techniques that power modern classifiers.

We begin with a broad overview of the text classification problem: how to frame a labeling task, choose a pipeline, and compare multiple algorithms on the same data. We then move from sparse bag-of-words features to dense word embeddings and document embeddings, exploring how distributional representations learned from large corpora can improve classification accuracy. The chapter concludes with an introduction to deep learning for text, including convolutional and recurrent architectures and a first look at how fine-tuned transformer models such as BERT achieve state-of-the-art performance.

Throughout, you will connect concepts from these notes to hands-on labs and practice notebooks. The progression from simple Naive Bayes classifiers to deep neural networks gives you both a practical toolkit and an understanding of the trade-offs involved in choosing a modeling approach.

In this chapter, you will learn to:

  • Frame a text labeling problem as a supervised classification task and identify appropriate evaluation metrics.

  • Build an end-to-end text classification pipeline: preprocessing, feature extraction, model training, and evaluation.

  • Compare Naive Bayes, Logistic Regression, and Support Vector Machine classifiers on a real-world imbalanced dataset.

  • Explain the intuition behind word embeddings (word2vec, GloVe) and use them as features for downstream classifiers.

  • Describe how Doc2Vec extends word embeddings to the document level and apply it to a multi-class problem.

  • Implement and interpret deep learning models (dense networks, CNNs, RNNs) for text classification.

  • Use a pre-trained BERT model for sentiment classification and contrast it with classical and shallow neural approaches.

In Chapter 1, we established what NLP is and how to acquire and preprocess text. In Chapter 2, we explored how to turn cleaned tokens into numerical vectors — one-hot encodings, bag-of-words, TF–IDF, word embeddings, and contextual BERT representations. This chapter moves from representations to modeling: given a numerical representation of text, how do we train a model that generalizes well to new documents? The techniques introduced here — classical classifiers, embedding-based models, and deep networks — form the backbone of the NLP modeling modules that follow.

These skills connect forward to:

  • Advanced NLP modules covering sequence labeling (Chapter 4), neural machine translation, and question answering.

  • LLM fine-tuning and prompting modules, where we will revisit BERT-style models and GPT-style decoders.

  • Agentic AI settings, where text classifiers and embedding models underpin retrieval, routing, and tool-using workflows.

The chapter is organized into five sections, each corresponding to a teaching session and accompanied by Jupyter-based labs and practice notebooks:

  1. Text classification: problem framing and pipeline design. We introduce the general text classification task, common benchmark datasets, and the standard pipeline: preprocessing → feature extraction → model training → evaluation. This section motivates the more detailed treatments in later sections.

  2. One pipeline, many classifiers. We work through a complete, reproducible pipeline on a real economic-news dataset, training and comparing Naive Bayes, Logistic Regression, and LinearSVC. Special attention is paid to class imbalance, evaluation metrics (accuracy, confusion matrix, ROC AUC), and the practical question of which model to deploy.

  3. Word embeddings for text classification. We revisit word embeddings from Chapter 2 in a classification context. We use the pre-trained Google News Word2Vec model to build sentence-level features by averaging word vectors, then train a logistic regression classifier on top.

  4. Document embeddings with Doc2Vec. We introduce Doc2Vec (Paragraph Vectors), which learns a single fixed-length vector per document without requiring aggregation of word vectors. We apply Doc2Vec to a six-class emotion dataset and discuss its strengths and limitations compared to word-averaging approaches.

  5. Deep learning for text classification. We introduce deep learning architectures for NLP: simple feedforward networks with embedding layers, convolutional networks (TextCNN), and recurrent networks (LSTM/GRU). We also show how a fine-tuned BERT model can be used for sentiment analysis.

Throughout the chapter, code examples accompany each section. Each section includes short, focused Python snippets that illustrate key workflows — such as building classification pipelines with scikit-learn, loading pretrained embeddings with gensim, training Doc2Vec models, and fine-tuning BERT with transformers.
