Preface#

What is this course about?#

This course provides a modern, practice-oriented introduction to Natural Language Processing (NLP), Large Language Models (LLMs), and Agentic AI systems. It is designed for data science and analytics students who want to understand how language technologies work, how to build reproducible NLP pipelines, and how to integrate LLMs and tools into larger decision-making workflows.

Rather than treating NLP, LLMs, and agents as separate topics, the course emphasizes their connections:

  • Foundational NLP skills for working with text data.

  • Classical and neural modeling techniques for core NLP tasks.

  • Transformer-based language models and prompt-driven workflows.

  • Tool-using and agentic systems that combine LLMs with external data and computation.

Throughout, we balance theory and practice by combining conceptual lectures with hands-on labs in Python, using real-world text from domains such as environmental science, public health, and policy.

Why NLP, LLMs, and Agentic AI?#

Language is central to how individuals, communities, and institutions describe the world, make decisions, and communicate uncertainty. NLP and LLMs provide computational tools to:

  1. Structure and summarize large volumes of text.
    Reports, scientific articles, policy documents, and conversations all contain rich information that can be made more accessible through classification, extraction, and summarization.

  2. Build data-driven decision-support systems.
    Textual signals can complement quantitative indicators in applications such as environmental monitoring, disaster response, and health communication.

  3. Design interactive systems that work with people, not just data.
    Agentic AI systems extend LLMs with tools and memory, enabling workflows that plan, call APIs, retrieve documents, and reason over multiple steps.

Understanding these technologies is increasingly important for responsible data science practice, especially when deploying models in real-world, high-impact contexts.

Topics and structure of the course#

The course is organized into eight main modules that build on each other:

  1. Introduction to NLP and NLP pipelines
    Foundations of text data, common NLP tasks, and end-to-end workflow design. We introduce corpora, tokenization, and the classic pipeline of data acquisition, preprocessing, modeling, and evaluation. You will see how small, well-designed examples connect to larger systems.

  2. Text preprocessing and representations
    Tokenization, normalization, bag-of-words, TF–IDF, embeddings, and transformer-based representations. This module covers how raw text becomes numerical data, contrasting classical sparse features with dense representations learned by neural models.

  3. Core NLP modeling
    Text classification, sequence labeling, sentiment analysis, and evaluation of supervised NLP pipelines. We build traditional supervised models on top of the representations introduced earlier and discuss metrics, error analysis, and model selection.

  4. Applied neural NLP
    Word2Vec, Doc2Vec, deep learning architectures, BERT, and practical implementation of end-to-end NLP workflows. Here you will implement neural models for standard tasks, compare them to classical approaches, and explore transfer learning with pretrained encoders.

  5. Introduction to LLMs
    Tokens, embeddings, transformer foundations, inference, and the role of LLMs in modern NLP systems. We study the high-level architecture of large language models, how they are trained, and how they differ from earlier neural models.

  6. Prompting, adaptation, and evaluation
    Prompt engineering, fine-tuning or adaptation strategies, task evaluation, and responsible use of LLMs. You will learn how to steer LLMs with prompts, when to adapt models with additional data, and how to evaluate LLM-based systems beyond accuracy.

  7. LLM applications and tool use
    Classification with LLMs, retrieval-oriented workflows, and practical integration of LLMs into data science pipelines. We focus on combining LLMs with retrieval, vector databases, and existing infrastructure to solve realistic problems.

  8. Agentic AI systems and final project integration
    Advanced text generation, tools, agents, LLM systems, and project design for real-world applications. The final module introduces agentic patterns—such as tool-calling, planning, and multi-step reasoning—and guides you in designing a capstone project that integrates course concepts.

Each module is supported by Jupyter-based lecture notes, runnable code examples, and labs that encourage experimentation with different modeling choices and evaluation strategies.

Computing environment: JupyterHub and Colab#

JupyterHub brings the power of notebook-based workflows to groups of users by providing each learner with a personal computational environment on shared infrastructure. It allows students, researchers, and data scientists to access pre-configured NLP and LLM environments without worrying about local installation or maintenance.

Key features of JupyterHub:

  1. Customizable – JupyterHub can serve a variety of environments and user interfaces (such as classic notebooks or JupyterLab), making it easy to provide a consistent setup for this course.

  2. Flexible – It supports pluggable authentication and controlled access, which is useful for managing course rosters and restricting resources to enrolled students.

  3. Scalable – JupyterHub can be deployed using modern container technology and Kubernetes, supporting anything from a small class to deployments with thousands of users.

  4. Portable – It is open-source and can run on commercial cloud providers, virtual machines, or institutional hardware, making it adaptable to different teaching and research contexts.

In this course, JupyterHub will be used to deliver a ready-to-use environment with the necessary libraries, datasets, and notebooks for NLP, LLMs, and agentic workflows, accessible through a web browser.

https://jupyter.org/hub

Google Colab, short for Google Colaboratory, is a cloud-based platform provided by Google that offers a free notebook environment to run and execute Python code in the browser. It can serve as a complementary environment for running course notebooks, experimenting with additional datasets, or continuing your work outside the institutional JupyterHub deployment.

Key features of Google Colab:

  1. Jupyter-style Notebooks

  2. Free Cloud Computing

  3. Collaboration

  4. Pre-installed Libraries

  5. Storage and Version Control

  6. Markdown Support

  7. Code Snippets

  8. Data Import and Export

https://colab.research.google.com/
