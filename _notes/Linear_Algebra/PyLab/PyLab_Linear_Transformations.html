8.5.1. Special Linear Transformations in \(\mathbb{R}^2\)#

8.5.1.1. Rotation#

Let \(R_\theta:~\mathbb{R}^2 \to \mathbb{R}^2\) be a linear transformation given by\linebreak rotating vectors through an angle of \(\theta\). Then the matrix \(A\) of \(R_\theta\) is given by

(8.1)#\[\begin{align} \begin{bmatrix}\cos( \theta ) & -\sin( \theta )\\\sin( \theta ) & \cos( \theta )\end{bmatrix}. \end{align}\]

Letâ€™s define the following rotation function.

# numpy
import numpy as np
from __future__ import division
from sympy import init_session, init_printing, Matrix
init_printing(use_unicode=True)

def Rot(t):
    return Matrix([[np.cos(t) , -np.sin(t)], [np.sin(t) , np.cos(t)]])

Example: Let \(R_\pi: ~\mathbb{R}^2 \to \mathbb{R}^2\) denote rotation through \(\pi\). Find the matrix of \(R_\pi\). Then, find \(B = R_\pi\begin{bmatrix}1 \\ 1\end{bmatrix}\), \(C = R_{\pi/2}\begin{bmatrix}1 \\ 1\end{bmatrix}\), and \(D = R_{-\pi/2}\begin{bmatrix}1 \\ 1\end{bmatrix}\)

A = Matrix([[1], [1]])
A

B = Rot(np.pi)*A
B

C = Rot(np.pi/2)*A
C

D = Rot(-np.pi/2)*A
D

# matplotlib
import matplotlib.pyplot as plt
%matplotlib inline
plt.style.use('seaborn-white')

fig, ax = plt.subplots(nrows=1, ncols=1, figsize=(6, 6))
_ = ax.scatter(A[0], A[1], marker='o', color='r', label='A')
_ = ax.scatter(B[0], B[1], marker='o', color='b', label='B')
_ = ax.scatter(C[0], C[1], marker='o', color='g', label='C')
_ = ax.scatter(D[0], D[1], marker='o', color='c', label='D')

_ = ax.set_xlabel('$x$', fontsize=18)
_ = ax.set_ylabel('$y$', fontsize=18)
_ = ax.legend(loc='best', bbox_to_anchor=(1.01, 0., 0.2, 0.6))
_ = ax.set_aspect('equal')
_ = ax.grid(True, which='both')
_ = ax.axhline(y=0, color='k')
_ = ax.axvline(x=0, color='k')

8.5.1.2. Reflection#

Let \(Q_m: ~\mathbb{R}^2 \to \mathbb{R}^2\) a linear transformation given by reflecting vectors over the line \(\vec{y} = m\,\vec{x}\) . Then the matrix of \(Q_m\) is given by

(8.2)#\[\begin{align} \frac{1}{1+m^2}\begin{bmatrix}1-m^2 & 2m\\2m &m^2-1\end{bmatrix} \end{align}\]

Now, we can define

def Ref(m):
    return Matrix([[1-m**2 , 2*m], [2*m , m**2-1]])/(1+m**2)

Example: Let \(Q_3: ~\mathbb{R}^2 \to \mathbb{R}^2\) denote reflection over the line \(y = 3x\) . Then \(Q_3\) is a linear transformation. Find the matrix of \(Q_3\). Then, find \(Q_3\begin{bmatrix}1 \\ 1\end{bmatrix}\).

Q3 = Ref(3)
Q3

A = Matrix([[1], [1]]) A

B = Q3*A
B

fig, ax = plt.subplots(nrows=1, ncols=1, figsize=(6, 6))
