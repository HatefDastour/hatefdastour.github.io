3.5. Creating New Geospatial Data#

Despite the abundance of existing geospatial data sources, there are numerous scenarios where the creation of new, custom geospatial data becomes necessary. These situations may arise due to several factors:

  1. Absence of data for specific geographic areas or phenomena [Goodchild, 2007]

  2. Requirement for more current or up-to-date information [Longley et al., 2015]

  3. Need for higher spatial resolution or improved accuracy compared to available datasets [Longley et al., 2015]

  4. Unique project requirements that cannot be met by existing data sources [Elwood, 2008]

The creation of new geospatial data enables researchers and practitioners to address gaps in existing datasets, verify or update outdated information, and develop bespoke datasets tailored to specific project needs. This process encompasses a range of methodologies, including field surveys, remote sensing techniques, digitization of analog maps, and derivation from existing digital data.

The subsequent sections will delve into various methods for generating new geospatial data, including:

  • Digitization of analog maps

  • Processing of remote sensing data

  • GPS data collection

  • Field surveys

  • Geocoding

  • Interpolation

  • Derivation from existing data

Each of these methods possesses unique strengths and is particularly suited to different data creation tasks. By gaining a comprehensive understanding of these techniques, GIS professionals can make informed decisions when selecting the most appropriate method for their specific data requirements.

3.5.1. Digitization of Analog Maps#

Digitizing analog maps is a crucial process in GIS that involves converting paper maps into digital formats suitable for use in GIS applications. This method is particularly valuable for preserving historical data, integrating legacy information with modern datasets, and updating outdated cartographic resources [Goodchild, 2007]. The digitization process typically involves three main steps:

  1. Scanning: The analog map is scanned at a high resolution (typically 300-600 dpi) to create a digital raster image. For example, a historical topographic map from the 1950s might be scanned to preserve its information in digital form [Rumsey et al., 2004].

  2. Georeferencing: The scanned image is imported into GIS software, where geographic coordinates are assigned to control points on the map. This process aligns the map with real-world coordinates, allowing it to be overlaid with other spatial data. For instance, a scanned city plan from the 19th century could be georeferenced to modern satellite imagery to study urban development over time [Jenny and Hurni, 2011].

  3. Digitizing: Features such as roads, boundaries, and landmarks are traced on-screen to create vector data layers. This process can be manual, where an operator traces features by hand, or semi-automated using feature extraction algorithms. For example, contour lines on a topographic map might be digitized to create a digital elevation model [El-Sheimy et al., 2005, Li et al., 2004].

Digitization serves several important purposes in geospatial analysis:

  • Historical Preservation: It allows for the digital archiving of rare or fragile maps, ensuring their information is preserved for future generations [Rumsey et al., 2004].

  • Data Integration: Digitized historical maps can be integrated with modern datasets, enabling temporal analysis of geographic phenomena. For instance, digitized land use maps from different periods can be compared to study urbanization patterns [Goodchild, 2007].

  • Data Updating: Outdated information on paper maps can be digitized and updated with current data. This is particularly useful for maintaining accurate cadastral records or updating infrastructure maps [Longley et al., 2015].

  • Accessibility: Digitization makes map information more accessible, allowing for easier sharing, analysis, and visualization in GIS environments [Goodchild, 2007].

While digitization is a powerful tool, it’s important to note that the process can introduce errors, particularly during georeferencing and manual digitizing. Therefore, quality control measures and an understanding of the original map’s limitations are crucial for ensuring the accuracy of the resulting digital data [Jenny and Hurni, 2011].

3.5.2. Processing of Remote Sensing Data#

Remote sensing involves acquiring data from satellite or aerial imagery, which can be processed to create various geospatial datasets. This process is essential for extracting valuable information from the vast amounts of data collected by remote sensing instruments such as satellites, drones, and aerial sensors.

3.5.2.1. Image Classification#

Image classification is a critical technique in remote sensing used to categorize pixels into land cover types. There are two primary methods:

  1. Supervised Classification: This method involves using training data, where the user provides known land cover types to guide the classification algorithm. It is particularly useful for specific applications where the land cover types are known in advance [Dastour and Hassan, 2023, Richards, 2022].

  2. Unsupervised Classification: This method does not require training data. Instead, it groups pixels into clusters based on their spectral properties. This is useful for exploratory analysis where the land cover types are not predefined [Lu et al., 2017, Romero et al., 2016].

For example, supervised classification can be used to map agricultural fields [Samaniego and Schulz, 2009, Xie et al., 2007], while unsupervised classification might be used to explore urban development patterns [Gašparović, 2020, Wang and Biljecki, 2022].

3.5.2.2. Feature Extraction#

Feature extraction involves identifying and extracting specific features such as roads, water bodies, and vegetation from remote sensing imagery. Advanced image processing techniques, including machine learning algorithms, are employed to enhance and interpret the data for feature extraction [Benediktsson et al., 2003, Romero et al., 2016]. For instance, extracting road networks from high-resolution satellite images can aid in urban planning and infrastructure development [].

3.5.2.3. Photogrammetry#

Photogrammetry is the science of making measurements from photographs, particularly for recovering the exact positions of surface points. It is used to create 3D models from overlapping images, which are invaluable for applications like topographic mapping and 3D city modeling. For example, photogrammetry can be used to generate digital elevation models (DEMs) from aerial photographs, which are crucial for flood risk assessment and infrastructure design [].

3.5.2.4. Applications and Importance#

Remote sensing data is invaluable for monitoring environmental changes, urban development, and natural resource management. It allows for:

  • Environmental Monitoring: Tracking changes in vegetation, water bodies, and land use under climate change scenarios [Dastour and Hassan, 2024, Dastour and Hassan, 2024].

  • Urban Development: Assisting in urban planning by providing up-to-date maps and models of urban areas [].

  • Natural Resource Management: Monitoring resources like forests and water bodies to ensure sustainable management and conservation [].

3.5.3. GPS Data Collection#

GPS technology allows for the accurate collection of geospatial data in the field:

  1. Point Data: Collecting specific locations such as landmarks or sampling sites.

  2. Line Data: Mapping linear features like roads or trails by recording a series of GPS points.

  3. Polygon Data: Defining area boundaries by walking the perimeter and recording GPS points.

GPS data is easily integrated into GIS software and can be used for navigation, mapping, and spatial analysis.

3.5.4. Field Surveys#

Field surveys involve collecting data directly from the field using various instruments:

  1. Total Stations: Measure angles and distances to create highly accurate maps.

  2. Theodolites: Used for measuring horizontal and vertical angles.

  3. Leveling Instruments: Determine elevation differences between points.

Field surveys are crucial for projects requiring high precision, such as cadastral mapping and engineering surveys.

3.5.5. Geocoding#

Geocoding is the process of converting addresses or place names into geographic coordinates:

  1. Address Matching: Using geocoding services to match addresses with their corresponding coordinates.

  2. Place Name Conversion: Translating place names into coordinates based on a geographic database.

Geocoding enables the mapping of tabular data with location information, such as customer addresses or service locations.

3.5.6. Interpolation#

Interpolation involves creating continuous surfaces from discrete point data:

  1. Inverse Distance Weighting (IDW): Estimates values based on the distance to nearby points.

  2. Kriging: A statistical method that provides estimates and measures of uncertainty.

  3. Natural Neighbor: Uses surrounding points to estimate values.

Interpolation is useful for creating elevation models, temperature maps, and pollution distribution maps.

3.5.7. Derivation from Existing Data#

New datasets can be created by analyzing and processing existing geospatial data:

  1. Buffering: Creating zones around features at specified distances.

  2. Overlay Analysis: Combining multiple datasets to identify relationships and patterns.

  3. Density Analysis: Calculating the concentration of features in an area.

These techniques allow for the generation of custom datasets that meet specific analytical needs.

By employing these methods, GIS professionals can create new geospatial data tailored to their project requirements, ensuring that they have the most relevant and accurate information for their analyses.
