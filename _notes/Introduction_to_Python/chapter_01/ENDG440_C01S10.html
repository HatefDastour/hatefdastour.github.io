1.10. Modules#

Modules in Python are files containing Python code, which can include functions, classes, and variables. They provide a way to organize and reuse code, making programs easier to manage and maintain. Modules can be built-in (part of the Python standard library), third-party (available via the Python Package Index, PyPI), or user-defined.

1.10.1. Importing Modules#

Modules are imported using the import statement. Once a module is imported, its functions, classes, and variables can be accessed using the dot (.) notation.

Example:

import math

result = math.sqrt(16)
print(result)  # Output: 4.0

4.0

In this example, the math module is imported, and its sqrt function is used to calculate the square root of 16.

1.10.2. Importing Specific Items from a Module#

Specific items from a module can be imported using the from ... import ... statement.

Example:

from math import pi, sin

print(pi)  # Output: 3.141592653589793
print(sin(pi / 2))  # Output: 1.0

3.141592653589793
1.0

Here, pi and sin are imported directly from the math module.

1.10.3. Aliasing Modules#

Modules can be aliased using the as keyword, allowing you to use a shorter or more convenient name.

Example:

import numpy as np

array = np.array([1, 2, 3, 4])
print(array)  # Output: [1 2 3 4]

[1 2 3 4]

In this example, the numpy module is imported as np.

1.10.4. User-Defined Modules#

User-defined modules are Python files created by the user. They can be imported in the same way as built-in or third-party modules.

1.10.4.1. Creating a module (mymodule.py):#

# mymodule.py

def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b

1.10.4.2. Importing and using the user-defined module:#

import mymodule

greeting = mymodule.greet("Alice")
print(greeting)  # Output: Hello, Alice!

sum_result = mymodule.add(5, 3)
print(sum_result)  # Output: 8

1.10.5. Importing All Items from a Module#

All items from a module can be imported using the from ... import * statement. However, this is generally discouraged as it can lead to namespace pollution and make the code harder to read.

Example:

from math import *

print(sqrt(16))  # Output: 4.0
print(pi)  # Output: 3.141592653589793

4.0
3.141592653589793

1.10.6. Module Search Path#

When a module is imported, Python searches for the module in the following order:

  1. The current directory.

  2. Directories in the PYTHONPATH environment variable.

  3. Standard library directories.

The module search path is stored in the sys.path variable.

Example:

import sys

print(sys.path)

1.10.7. Reloading Modules#

Modules are imported only once per session. To reload a module (useful during development), the importlib.reload() function can be used.

Example:

import mymodule
from importlib import reload

# Modify mymodule.py externally here

reload(mymodule)
