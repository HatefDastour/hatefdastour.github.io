Data Smoothing¶

  • Savitzky-Golay Filter

Data Smoothing is a tool that eliminates noise from a dataset. To exlain this, consider $f(x)=\sin(x)\cos(x)$ with $0\leq x \leq 4\pi$. We can discretize $[0,4\pi]$ using 100 points. We have

In [2]:
## Data
x = np.linspace(0, 4*np.pi, 100)
y = np.sin(x)*np.cos(x)

Adding noise to the data using numpy.random.normal.

In [3]:
## Noisy Data
mu, sigma = 0, 0.1 # mean and standard deviation
noise = np.random.normal(mu, sigma, len(x))
y_noise = y + noise

Savitzky-Golay Filter¶

Without getting into the details, we only apply SavitzkyGolay filter here and interested readers can see this link for more details. We use scipy.signal.savgol_filter from scipy package. The details on the usage of this package can be found here.

In [4]:
## Filtering noisy data
y_filtered=signal.savgol_filter(y_noise,23,4)
