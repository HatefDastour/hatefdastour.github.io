4. Drawing Functions#

4.1. line#

line function can be used for drawing a line segment connecting two points.

cv2.line(img, pt1, pt2, color[, thickness[, lineType[, shift]]]) → None

Parameters:

  • img – Image.

  • pt1 – First point of the line segment.

  • pt2 – Second point of the line segment.

  • color – Line color.

  • thickness – Line thickness.

  • lineType –

  • Type of the line:

    • 8 (or omitted) - 8-connected line.

    • 4 - 4-connected line.

    • CV_AA - antialiased line.

  • shift – Number of fractional bits in the point coordinates.

Example:

We use the sklearn image dataset to demonstrate basic operations using Open CV.

from sklearn.datasets import load_sample_image, load_sample_images
import matplotlib.pyplot as plt
from matplotlib.font_manager import FontProperties

Images = load_sample_images()
Names = [x.split("\\")[-1].replace('.jpg','').title() for x in Images['filenames']]
Images = Images['images']

def ImShow(Images, Names, Title = 'Images', axis_tight = False):
    fig, ax = plt.subplots(1, 2 , figsize = (17, 6))
    ax = ax.ravel()
    font = FontProperties()
    font.set_weight('bold')
    for i in range(2):
        _ = ax[i].imshow(Images[i])
        if axis_tight:
            _ = ax[i].axis('tight')
        _ = ax[i].set_aspect(1)
        _ = ax[i].axis('off')
        _ = ax[i].set_title(Names[i], fontproperties=font, fontsize = 16)
    _ = fig.subplots_adjust(wspace= 0.01)
    if Title:
        _ = fig.suptitle(Title, fontproperties=font, fontsize = 18)
    return fig, ax
# Image Show
_,_ = ImShow(Images, Names, Title = 'Original Images')

Here’s a breakdown of the code:

  1. Importing the necessary libraries:

    • sklearn.datasets.load_sample_image and sklearn.datasets.load_sample_images are functions from the scikit-learn library that allow you to load sample images.

    • matplotlib.pyplot is a plotting library used for visualizations.

    • matplotlib.font_manager.FontProperties is a class that handles font properties for text rendering in matplotlib.

  2. Loading sample images:

    • The Images variable is assigned the result of calling load_sample_images(), which loads a set of sample images.

    • The Names variable is created by iterating over the filenames of the loaded images, extracting the title from each filename.

  3. Defining the ImShow function:

    • The function takes three parameters: Images (the images to display), Names (the corresponding names of the images), and optional Title and axis_tight parameters.

    • It creates a figure and axes using plt.subplots.

    • The images and their titles are displayed in a loop.

    • The axis_tight parameter determines whether to adjust the axis limits tightly around the image.

    • The set_aspect(1) ensures that the aspect ratio of the image is preserved.

    • The axes are turned off using axis('off').

    • The titles of the images are set using set_title.

    • The spacing between the subplots is adjusted using subplots_adjust.

    • If a Title is provided, it is set as the figure’s suptitle.

  4. Calling ImShow to display the images:

    • The ImShow function is called with the Images, Names, and Title parameters set.

    • The return values (fig and ax) are assigned to _ to ignore them.

The result is a figure displaying two images side by side, with their corresponding titles. The figure’s overall title is set as “Original Images”.

4.2. Rectangle#

rectangle draws a simple, thick, or filled up-right rectangle.

cv2.rectangle(Image, pt1, pt2, color[, thickness[, lineType[, shift]]]) → None

Parameters:

  • pt1 – Vertex of the rectangle.

  • pt2 – Vertex of the rectangle opposite to pt1 .

  • color – Rectangle color or brightness (grayscale image).

  • thickness – Thickness of lines that make up the rectangle. Negative values, like CV_FILLED , mean that the function has to draw a filled rectangle.

  • lineType – Type of the line. See the line() description.

  • shift – Number of fractional bits in the point coordinates.

_ = ax[1].set_title('Image with a Rectangle Area', fontproperties=font, fontsize = 12)

del i

4.3. Circle#

circle draws a circle.

cv2.circle(Img, center, radius, color[, thickness[, lineType[, shift]]]) → None

Parameters:

  • center – Center of the circle.

  • radius – Radius of the circle.

  • color – Circle color.

  • thickness – Thickness of the circle outline, if positive. Negative thickness means that a filled circle is to be drawn.

  • lineType – Type of the circle boundary. See the line() description.

  • shift – Number of fractional bits in the coordinates of the center and in the radius value.

Example:

Img = Images[0].copy()
Out = Img.copy()

_ = cv2.circle(Out, (220, 220), 200, Colors['Yellow'], thickness=2, lineType=cv2.LINE_AA)

fig, ax = plt.subplots(1, 2 , figsize = (17, 6))
ax = ax.ravel()
_ = fig.suptitle('Circle', fontproperties=font, fontsize = 18)

_ = ax[0].imshow(Img)
_ = ax[0].set_title('Original Image', fontproperties=font, fontsize = 12)
_ = ax[1].imshow(Out)
_ = ax[1].set_title('Image with a Circle Area', fontproperties=font, fontsize = 12)
for i in range(2):
    _ = ax[i].set_aspect(1)
    _ = ax[i].axis('off')
del i
del Img, Out

Let’s break down the code step by step:

  1. Img = Images[0].copy(): This line creates a copy of the first image in the Images list and assigns it to the variable Img. The .copy() function ensures that any modifications made to Img do not affect the original image.

  2. Out = Img.copy(): This line creates another copy of Img and assigns it to the variable Out. Again, this is done to keep the original image intact while performing modifications on Out.

  3. _ = cv2.circle(Out, (220, 220), 200, Colors['Yellow'], thickness=2, lineType=cv2.LINE_AA): This line adds a circle to the image Out. The parameters of the cv2.circle function are as follows: the first parameter is the image on which the circle will be drawn (Out in this case), the second parameter is the center coordinates of the circle (in this case, (220, 220)), the third parameter is the radius of the circle (200 in this case), the fourth parameter is the color of the circle (specified as Colors['Yellow']), and the remaining parameters control the thickness and line type of the circle.

  4. fig, ax = plt.subplots(1, 2 , figsize = (17, 6)): This line creates a figure with two subplots side by side using plt.subplots. The first parameter specifies the number of rows (1) and the second parameter specifies the number of columns (2) of subplots. The figsize parameter sets the size of the figure.

  5. ax = ax.ravel(): This line flattens the ax array of subplots into a 1-dimensional array. It allows accessing each subplot using a single index.

  6. _ = fig.suptitle('Circle', fontproperties=font, fontsize = 18): This line adds a super title to the figure, displaying the text ‘Circle’ with a specified font and fontsize.

  7. _ = ax[0].imshow(Img): This line displays the original image Img on the first subplot.

  8. _ = ax[0].set_title('Original Image', fontproperties=font, fontsize = 12): This line sets the title of the first subplot as ‘Original Image’ with a specified font and fontsize.

  9. _ = ax[1].imshow(Out): This line displays the modified image Out (with the added circle) on the second subplot.

  10. _ = ax[1].set_title('Image with a Circle Area', fontproperties=font, fontsize = 12): This line sets the title of the second subplot as ‘Image with a Circle Area’ with a specified font and fontsize.

  11. for i in range(2):: This line starts a loop that iterates twice, corresponding to the two subplots.

  12. _ = ax[i].set_aspect(1): This line sets the aspect ratio of each subplot to 1, ensuring that the image is displayed without distortion.

  13. _ = ax[i].axis('off'): This line turns off the axis labels and ticks for each subplot, removing the coordinate axes.

  14. del i: This line deletes the loop variable i to clean up the namespace.

  15. del Img, Out: This line deletes the variables Img and Out to free up memory.

Overall, the code creates a figure with two subplots

4.4. Ellipse#

ellipse draws a simple or thick elliptic arc or fills an ellipse sector.

cv2.ellipse(img, center, axes, angle, startAngle, endAngle, color[, thickness[, lineType[, shift]]]) → None

Parameters:

  • img – Image.

  • center – Center of the ellipse.

  • axes – Half of the size of the ellipse main axes.

  • angle – Ellipse rotation angle in degrees.

  • startAngle – Starting angle of the elliptic arc in degrees.

  • endAngle – Ending angle of the elliptic arc in degrees.

  • box – Alternative ellipse representation via RotatedRect or CvBox2D. This means that the function draws an ellipse inscribed in the rotated rectangle.

  • color – Ellipse color.

  • thickness – Thickness of the ellipse arc outline, if positive. Otherwise, this indicates that a filled ellipse sector is to be drawn.

  • lineType – Type of the ellipse boundary. See the line() description.

  • shift – Number of fractional bits in the coordinates of the center and values of axes.

Example:

Img = Images[0].copy()
Out = Img.copy()
_ = cv2.ellipse(Out, (220, 220), (200, 180), 45, 0, 360, Colors['Red'], thickness=2, lineType=cv2.LINE_AA)

fig, ax = plt.subplots(1, 2 , figsize = (17, 6))
ax = ax.ravel()
_ = fig.suptitle('Circle', fontproperties=font, fontsize = 18)

_ = ax[0].imshow(Img)
_ = ax[0].set_title('Original Image', fontproperties=font, fontsize = 12)
_ = ax[1].imshow(Out)
_ = ax[1].set_title('Image with a Ellipse Area', fontproperties=font, fontsize = 12)
for i in range(2):
    _ = ax[i].set_aspect(1)
    _ = ax[i].axis('off')
del i
del Img, Out

Let’s go through it step by step:

  1. It creates a copy of the first image in the Images list and assigns it to the variable Img.

  2. It creates another copy of Img and assigns it to the variable Out.

  3. It uses the OpenCV (cv2) library to draw an ellipse on the Out image. The ellipse is centered at coordinates (220, 220), has axes lengths of 200 and 180, is rotated by 45 degrees, and spans a full circle (0 to 360 degrees). The ellipse is drawn in the color specified by Colors['Red'], with a thickness of 2 pixels, using anti-aliased lines (cv2.LINE_AA).

  4. It creates a figure with two subplots using plt.subplots(). The figure size is set to (17, 6).

  5. It assigns the subplots to the variables fig and ax.

  6. It sets the title of the figure to ‘Circle’ using fig.suptitle().

  7. It displays the original image (Img) in the first subplot (ax[0]) using ax[0].imshow().

  8. It sets the title of the first subplot to ‘Original Image’ using ax[0].set_title().

  9. It displays the modified image (Out) in the second subplot (ax[1]) using ax[1].imshow().

  10. It sets the title of the second subplot to ‘Image with an Ellipse Area’ using ax[1].set_title().

  11. It iterates over the two subplots (range 0 to 1) and performs the following operations for each subplot:

    • Sets the aspect ratio of the subplot to 1 using ax[i].set_aspect(1).

    • Turns off the axis labels using ax[i].axis('off').

  12. It deletes the variable i, Img, and Out from the namespace to clean up memory.

Overall, this code generates a figure with two subplots. The first subplot shows the original image, and the second subplot shows the same image with an ellipse drawn on it. The ellipse is drawn using OpenCV’s cv2.ellipse() function.

4.5. clipLine#

clipLine clips the line against the image rectangle..

cv2.clipLine(imgRect, pt1, pt2) → retval, pt1, pt2

Parameters:

  • imgSize – Image size. The image rectangle is Rect(0, 0, imgSize.width, imgSize.height) .

  • imgRect – Image rectangle.

  • pt1 – First line point.

  • pt2 – Second line point.

Img = Images[0].copy()
Out = Img.copy()

_ = cv2.rectangle(Out, (50, 20), (380, 400), Colors['Green'], 2)
_ = ret, p1, p2 = cv2.clipLine((0, 0, Out.shape[0], Out.shape[1]), (50, 20), (380, 400))

if ret:
    cv2.line(Out, p1, p2, Colors['Red'], 3)

fig, ax = plt.subplots(1, 2 , figsize = (17, 6))
ax = ax.ravel()
_ = fig.suptitle('Circle', fontproperties=font, fontsize = 18)

_ = ax[0].imshow(Img)
_ = ax[0].set_title('Original Image', fontproperties=font, fontsize = 12)
_ = ax[1].imshow(Out)
_ = ax[1].set_title('Image with a Rectangle Area', fontproperties=font, fontsize = 12)
for i in range(2):
    _ = ax[i].set_aspect(1)
    _ = ax[i].axis('off')
del i

4.6. getTextSize#

getTextSize calculates the width and height of a text string.

cv2.getTextSize(text, fontFace, fontScale, thickness) → retval, baseLine

Parameters:

  • text – Input text string.

  • text_string – Input text string in C format.

  • fontFace – Font to use. See the putText() for details.

  • fontScale – Font scale. See the putText() for details.

  • thickness – Thickness of lines used to render the text. See putText() for details.

  • baseLine – Output parameter - y-coordinate of the baseline relative to the bottom-most text point.

  • baseline – Output parameter - y-coordinate of the baseline relative to the bottom-most text point.

  • font – Font description in terms of old C API.

  • text_size – Output parameter - The size of a box that contains the specified text.

4.7. putText#

putText draws a text string.

cv2.putText(img, text, org, fontFace, fontScale, color[, thickness[, lineType[, bottomLeftOrigin]]]) → None

Parameters:

  • img – Image.

  • text – Text string to be drawn.

  • org – Bottom-left corner of the text string in the image.

  • font – CvFont structure initialized using InitFont().

  • fontFace – Font type. One of FONT_HERSHEY_SIMPLEX, FONT_HERSHEY_PLAIN, FONT_HERSHEY_DUPLEX, FONT_HERSHEY_COMPLEX, * FONT_HERSHEY_TRIPLEX, FONT_HERSHEY_COMPLEX_SMALL, FONT_HERSHEY_SCRIPT_SIMPLEX, or FONT_HERSHEY_SCRIPT_COMPLEX, where each of the font ID’s can be combined with FONT_ITALIC to get the slanted letters.

  • fontScale – Font scale factor that is multiplied by the font-specific base size.

  • color – Text color.

  • thickness – Thickness of the lines used to draw a text.

  • lineType – Line type. See the line for details.

  • bottomLeftOrigin – When true, the image data origin is at the bottom-left corner. Otherwise, it is at the top-left corner.
