5. Image Thresholding#

5.1. Simple Thresholding#

Applies a fixed-level threshold to each array element. It can be done as follows,

cv2.threshold(Source, threshold value, maximum value, type[, Output]) → retval, Output

Parameters:

  • Source: input array (single-channel, 8-bit or 32-bit floating point).

  • Output: output array of the same size and type as Source.

  • threshold value: threshold value.

  • maximum value – maximum value to use with the THRESH_BINARY and THRESH_BINARY_INV thresholding types.

  • type:

    • THRESH_BINARY

    • THRESH_BINARY_INV

    • THRESH_TRUNC

    • THRESH_TOZERO

    • THRESH_TOZERO_INV

5.1.1. THRESH_BINARY#

\[\begin{split}\text{Output }(x,y) = \begin{cases}\text{maximum value}&\mbox{if source(x,y) > threshold value},\\0,&\mbox{otherwise}\end{cases}\end{split}\]

Example:

We use the sklearn image dataset to demonstrate basic operations using Open CV.

from sklearn.datasets import load_sample_image
Img = load_sample_image('flower.jpg')

import cv2
import matplotlib.pyplot as plt
from matplotlib.font_manager import FontProperties

_ , Out = cv2.threshold(Img, 130, 255,cv2.THRESH_BINARY)

fig, ax = plt.subplots(1, 2 , figsize = (17, 6))
ax = ax.ravel()
font = FontProperties()
font.set_weight('bold')
_ = fig.suptitle('THRESH_BINARY', fontproperties=font, fontsize = 18)

_ = ax[0].imshow(Img)
_ = ax[0].set_title('Original Image', fontproperties=font, fontsize = 12)
_ = ax[1].imshow(Out)
_ = ax[1].set_title('Threshold value = 130,  Maximum value = 255', fontproperties=font, fontsize = 12)
for i in range(2):
    _ = ax[i].set_aspect(1)
    _ = ax[i].axis('off')
del i

Let’s break down the code step by step:

  1. Import necessary libraries:

    • sklearn.datasets.load_sample_image: This function is used to load a sample image from the scikit-learn dataset. It is used to demonstrate image processing techniques.

    • cv2: This is the OpenCV library, which is widely used for computer vision tasks, including image processing and manipulation.

    • matplotlib.pyplot: This library is used for creating visualizations and plotting.

    • matplotlib.font_manager.FontProperties: This is used to set the properties of the font used in the plot.

  2. Load the sample image:

    • The code loads a sample image called “flower.jpg” using the load_sample_image function from scikit-learn’s dataset.

  3. Apply binary thresholding to the image:

    • The cv2.threshold function is used to apply binary thresholding to the loaded image (Img). Binary thresholding is a simple image segmentation technique where pixels with intensity values above a certain threshold are set to a maximum value (here, 255) and the rest are set to 0. In this case, the threshold value is set to 130.

  4. Create a figure and subplot for displaying the images:

    • plt.subplots(1, 2, figsize=(17, 6)): This creates a figure with one row and two columns of subplots, with a size of 17x6 inches.

    • ax = ax.ravel(): The subplots are flattened into a 1D array (ax) for easier access.

  5. Set the font properties and create the title:

    • fig.suptitle: This sets the main title of the entire figure.

    • ax[0].set_title and ax[1].set_title: These set the titles of the two subplots.

  6. Display the images in the subplots:

    • ax[0].imshow(Img): This displays the original image in the first subplot.

    • ax[1].imshow(Out): This displays the thresholded image in the second subplot.

  7. Set aspect ratio and turn off axis for both subplots:

    • ax[i].set_aspect(1): This sets the aspect ratio of the images in the subplots to be equal, preventing distortion.

    • ax[i].axis('off'): This turns off the axis for each subplot, so no axes or ticks are displayed.

  8. Clean up:

    • del i: The loop variable i is deleted to avoid any potential confusion if it is used later in the code.

Overall, this code snippet demonstrates how to load an image, apply binary thresholding, and display the original and thresholded images using Python libraries.

5.2. AdaptiveThresholding#

Applies an adaptive threshold threshold to an array.. It can be done as follows,

cv2.adaptiveThreshold(Source, Maximum Value, Adaptive Method, thresholdType, blockSize, C[, Output]) → Output

Parameters:

  • Source: Source 8-bit single-channel image.

  • Output: Destination image of the same size and the same type as Source.

  • Maximum Value – Non-zero value assigned to the pixels for which the condition is satisfied. See the details below.

  • Adaptive Method: Adaptive thresholding algorithm to use, ADAPTIVE_THRESH_MEAN_C or ADAPTIVE_THRESH_GAUSSIAN_C . See the details below.

  • thresholdType:

    • THRESH_BINARY

    • THRESH_BINARY_INV

  • blockSize: Size of a pixel neighborhood that is used to calculate a threshold value for the pixel: 3, 5, 7, and so on.

  • C: Constant subtracted from the mean or weighted mean (see the details below). Normally, it is positive but may be zero or negative as well.
