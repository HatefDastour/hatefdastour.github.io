4. Mini-Lab: Hands-On Lab#

Part 4 synthesizes Parts 1-3 through 15-minute practical exercises on real academic tasks.

Lab Objectives

  • Select framework + technique combinations

  • Build/execute/verify complete prompts

  • Compare naive vs structured outputs

4.1. Task Categories#

Select one from your current workload:

  • Scaffold assignment → ROSES + CoT

  • Student feedback → CREATE + Few-Shot

  • Active-learning quiz → ERA

  • Screen abstracts → CARE + Few-Shot

  • Grant Narrative Polish → OCEAN + Chain-of-Thought

  • Data Categorization (Qualitative) → AIM + Few-Shot

  • Abstract revision → CREATE

  • Lit review synthesis → CARE

  • Grant narrative → OCEAN + Few-Shot

  • Committee timeline → ROSES + CoT

  • Policy tiers → OCEAN

  • Email series → ERA + Few-Shot

4.2. 4-Step Process (15 min)#

Step 1: Naive Prompt (2 min)
Write first instinct (no framework):

“Write a rubric for my capstone.”

Step 2: Framework Rebuild (5 min)
Use Section 2 framework + technique.

Template:

[Framework]

[CoT/Few-Shot]: “Think step-by-step…” OR examples

Constraints: [format/length/verify]

Step 3: Compare (4 min)
Test in ChatGPT/Claude/Gemini/Perplexity/etc. Note improvements.

Step 4: Table Share (4 min)

  1. Framework + technique

  2. Key improvement

  3. Remaining concerns

4.3. Example: Rubric Design#

Naive:

“Write a rubric for my capstone project.”

OCEAN + CoT:

Objective: Analytic rubric for 20-page senior capstone
Context: [Major] seniors, distinguish Proficient/Exemplary
Action: Think step-by-step:

  1. Define 5 criteria (Thesis/Methods/etc)

  2. 4 levels each (Exemplary→Beginning)

  3. Summary feedback template
    Needs: Table format, critical thinking focus

Typical Gains

✓ Specific criteria ✓ Clear levels ✓ Context-fit
⚠ Still needs: discipline terms, local context, citation verification
