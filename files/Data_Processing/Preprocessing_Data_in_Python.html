Dealing with Missing Data¶

We can use the [SimpleImputer] for imputing missing values. This can be done by a provided constant value or using the statistics (mean, median or mode (most frequent)) of each column. Here, we are going to implement SimpleImputer with the mean strategy.

In [107]:
imp_mean = SimpleImputer(missing_values=np.nan, strategy='mean')
## imputing the data
X=imp_mean.fit_transform(dfC.iloc[:,:-1].values)
Y=imp_mean.fit_transform(dfC.iloc[:,-1].values.reshape(-1,1))
